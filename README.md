This repository contains a Jupyter Notebook analyzing bike trip data from Divvy for Q3+Q4 2019. The notebook includes data importing, cleaning, transformation, exploration, and visualization tasks.

## Requirements
- Python 3.x
- pandas
- matplotlib

## Getting Started
1. Clone the repo
2. Open the Jupyter Notebook file and run each cell.

# Day 1:

## Features
- Reads multiple quarterly CSV files and merges them.
- Adds calculated fields like ride length and weekdays.
- Grouping and summarizing user types.
- Visualizes data distributions with pie charts and other plots.
